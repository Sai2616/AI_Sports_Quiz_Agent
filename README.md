# 🏆 AI Sports Quiz Generator

An AI-powered Sports Quiz Generator built using **React, FastAPI, ChromaDB, Tavily Search, and Google Gemini AI**. The application generates dynamic multiple-choice quizzes using Retrieval-Augmented Generation (RAG) by combining local sports knowledge with real-time web search.

---

## 🚀 Features

- 🎯 Generate AI-powered sports quizzes
- 📚 Retrieval-Augmented Generation (RAG)
- 🗂️ ChromaDB vector database for semantic search
- 🌐 Tavily Web Search for latest sports information
- 🤖 Google Gemini AI for quiz generation
- 📊 Interactive quiz interface
- ✅ Score calculation
- 📈 Progress bar
- 💡 Answer explanations
- 🔄 Responsive React frontend
- ⚡ FastAPI backend
